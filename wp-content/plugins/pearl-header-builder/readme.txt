Pearl Header Builder gives you complete freedom to compose a header that perfectly suits your site.
 
== Description ==
 
Freely customize your theme header and build responsive headers for smartphones, tablets, and desktops.
With an intuitive drag-n-drop editor it’s easy to build a customized header and provide great user experience on all devices and platforms.
 
[Watch the Plugin Demo here](http://hb.stylemixthemes.com/)

https://www.youtube.com/watch?v=_WTp-6E762A
 
The Pearl Header Builder originally integrated into the [Pearl](https://themeforest.net/item/pearl-true-multiniche-wordpress-theme/20432158?ref=stylemixthemes) – multi-industry business WordPress theme by [StylemixThemes](https://stylemixthemes.com/).

== Installation ==
This section describes how to install the plugin and get it working.
 
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Header Builder -> to configure the plugin
4. Include a header builder shortcode into your theme header.php
4. Please find more details on Plugin Installation in documentation
